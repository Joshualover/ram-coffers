# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in ram-coffers, please report it to scott@elyanlabs.ai.

## Response Timeline

- **Acknowledgment**: Within 24 hours
- **Initial Response**: Within 72 hours
- **Resolution**: Depends on severity

## Security Best Practices

- Review NUMA topology calculations for potential overflow
- Validate weight inputs in banking operations
- Keep system dependencies updated

## Supported Versions

| Version | Supported |
|---------|-----------|
| main    | ✅        |

Thank you for helping keep ram-coffers secure!
